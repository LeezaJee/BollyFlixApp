BollyFlix Documentation

Objective

Creation of the server-side component of a “movies” web application specialized in Bollywood movies.

The web application will provide users with access to information about different movies, directors, actors and genres. User will also be able to sign up, update their personal information, add movies to their favorites list and remove their account if they wish to.

REQUESTS

Request URL Method Request Data Format Response Data Format
get list of all movies /movies GET none JSON object with information about all movies
get movie by title /movies/:title GET none JSON object with information about a specific movie
get movies by genre /movies/genres/:genreName GET none JSON object with movies according to requested genre
get directors info /movies/directors/:directorName GET none JSON object with information about a movie director
get movies by actor /movies/actors/:actorName GET none JSON object with movies according to actor's name
get all users /users GET none JSON object with information about all users
create a new user /users POST JSON object with info about a new user: { "name": "Harry", "favoriteMovies": "Badla"} JSON object with user info and ID: {"name": "Harry", "favoriteMovies": "Badla", "Id": "1da28a33-b4d2-43af-9385-6546f306f448"}
update user info /users/:id PUT JSON object containing all required fields like: {"name": "new Name", "favoriteMovies": "Mimi"} JSON object with updated user information
add a movie to favorites /users/:id/:movieTitle POST none success that movie has been added or error message if not
delete movie from favorites /users/:id/:movieTitle DELETE none success message that movie has been removed from favorites or error message
delete a user /users/:id DELETE none success message that user has been removed or error message if not
