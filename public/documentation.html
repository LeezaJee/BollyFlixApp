BollyFlix Documentation

Objective

Creation of the server-side component of a “movies” web application specialized in Bollywood movies.

The web application will provide users with access to information about different movies, directors, actors and genres. User will also be able to sign up, update their personal information, add movies to their favorites list and remove their account if they wish to.

REQUESTS

Request URL HTTP Method Request Data Format RESPONSE DATA BODY FORMAT
